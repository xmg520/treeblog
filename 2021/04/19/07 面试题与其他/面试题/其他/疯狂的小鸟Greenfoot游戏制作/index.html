疯狂的小鸟Greenfoot游戏制作

Mzx 2021-04-19 10:26:26
Categories： Tags：

Greenfoot介绍

Greenfoot，是一个java集成开发环境（IDE、Integrated Development Environment），Greenfoot需要JDK的支持。

Greenfoot 为编程人员提供了丰富的API （应用程序接口），通过调用这些API 可以方便地编写各种游戏操作。

Greenfoot 提供了几 个类： World 类、Actor 类、Greenfoo 类、GreenfootImage 类、MouseInfo 类、GreenfootSound 类、UserInfo类。
World 类和 Actor 类是抽象类，不能被直接使用，在设计程序时要创建它们的子类；World 类和 Actor 类分别作为游戏世界（场景）和游戏角色的父类， World是一个承载Actor游戏对象们的世界（场景）。它本身也是一个二维的坐标格网络。

在Greenfoot世界中，每一个物体（对象 Object）都称为一个 Actor。每个Actor都拥有一个相对于世界的坐标，和一个用于显示的外观（也就是：一个图标）。

学习项目：疯狂的小鸟

设置界面为600*400比例为1:1,初始页面添加Flappybird、Pipe、Score标签

/**

 * 
 * @author (your name) 
 * @version (a version number or a date)
 */
public class MyWorld extends World
{

    /**
     * Constructor for objects of class MyWorld.
     * 
     */
    public MyWorld()
    {    
        // Create a new world with 600x400 cells with a cell size of 1x1 pixels.
        super(600, 400, 1); 
        addObject(new Flappybird(), 100, 300);
        addObject(new Pipe(), 300, 150);
        addObject(new Pipe(), 600, 200);
        addObject(new Score(), 300, 100);
        
    }
}

Actor类

所有游戏角色都要继承Actor抽象类，其中act方法是程序运行时角色类运行的方法

Flappybird类

import greenfoot.*;  // (World, Actor, GreenfootImage, Greenfoot and MouseInfo)
/**
 * Write a description of class Flappybird here.
 * setRotation((int)(1 * 16)); Greenfoot.playSound("flay.mp3");
 * @author (your name) 
 * @version (a version number or a date)
 */
public class Flappybird extends Actor
{
    private boolean isbg = false;
    private double g = 1; 
    private int y = 300;
    private boolean haspressed = false;
    private boolean isalive = true;  
    private boolean isacross = false;
    private boolean hasaddscore = false; //初始值表示还没加过分
    public Flappybird(){
        GreenfootImage image = getImage();
        image.scale(50, 40);
    }
    /**
     * Act - do whatever the Flappybird wants to do. This method is called whenever
     * the 'Act' or 'Run' button gets pressed in the environment.
     */
    public void act() 
    {
        //如果按下空格，y坐标减小，实现向上飞
        if(spacePressed()){
            g=-2;//数字越小飞的越高
        }
        g +=0.1; //g的值每次都会增加0.1
        y += g; //这样y的值就不是匀速变化的，而是越来越大
        setLocation(getX(), (int)(y));
        //碰到管道游戏游戏结束
        if(isTouchpipe()){
            isalive = false;
        }
        //碰到边缘游戏结束
        if(isAtEdge()){
            Greenfoot.playSound("peng.mp3");
            isalive = false;
        }
        //游戏结束
        if(!isalive){
            getWorld().addObject(new Gameover(), 300, 200);
            getWorld().removeObject(this);
        }
        //加分判断
        if(!hasaddscore && isacross && isalive){
            Greenfoot.playSound("score.mp3");
            Score.add(1);
        }
        hasaddscore = isacross;
        // Add your action code here.
    }
    //返回空格键是否按下
    public boolean spacePressed(){
        boolean pressed = false;
        if(Greenfoot.isKeyDown("space")){
            if(!isbg){
                Greenfoot.playSound("by.mp3");
                isbg = true;
            }
            if(!haspressed){//如果一直没有松开空格键，不再返回true
                Greenfoot.playSound("flay.mp3");
                pressed = true;
            }
            haspressed = true; //已经按下了空格键
        }else{
            haspressed = false;
        }
        return pressed;
    }
    //碰撞检测
    public boolean isTouchpipe(){
        isacross = false;
        for(Pipe pipe : getWorld().getObjects(Pipe.class)){
            if(Math.abs(pipe.getX() - getX()) < 60 ){
                if(Math.abs(pipe.getY() + 30 - getY()) > 37){
                    Greenfoot.playSound("peng.mp3");
                    isalive = false;
                }
                isacross = true; //水平方向已经发生了碰撞
            }
        }
        return !isalive;
    }
}

Score类

定义初始量score记分，定义add方法在用户达成特定条件时加分，同时在act方法中展示到页面

在World类加载时创建

import greenfoot.*;  // (World, Actor, GreenfootImage, Greenfoot and MouseInfo)

/**
 * Write a description of class Score here.
 * 
 * @author (your name) 
 * @version (a version number or a date)
 */
public class Score extends Actor
{
    /**
     * Act - do whatever the Score wants to do. This method is called whenever
     * the 'Act' or 'Run' button gets pressed in the environment.
     */
    public static int score;//计分变量
    public Score(){
        score = 0;//初始化为0
    }
    public void act(){
        World myWorld = getWorld();
        myWorld.showText(String.valueOf(score), 300, 100);//在世界中显示变量值
    }
    //计分操作
    public static void add(int num){
        score += num;
    }
}
